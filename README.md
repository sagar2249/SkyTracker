# 🌤️ SkyTrack – Advanced Weather Forecasting App

SkyTrack is a modern and responsive weather application that provides real-time weather updates, hourly forecasts, and 5-day predictions for any location worldwide. It offers a clean UI along with useful weather insights and alerts.

---

## 🚀 Features

* 🌍 Search weather by city name
* 📍 Get weather using current location
* 🌡️ Real-time temperature and conditions
* 📊 Detailed metrics (humidity, wind speed, pressure, visibility, feels like)
* ⏱️ Hourly forecast (next 24 hours)
* 📅 5-day weather forecast
* ⚠️ Severe weather alerts (thunderstorm, heat, wind, etc.)
* 🔔 Browser notification support
* 🔄 Auto-refresh every 10 minutes
* 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* OpenWeatherMap API

## ⚙️ Setup Instructions

1. Clone the repository:

```
git clone https://github.com/your-username/skytrack.git
```

2. Navigate to the project folder:

```
cd skytrack
```

3. Open `app.js` and add your API key:

```js
const API_KEY = 'YOUR_API_KEY';
```

4. Run the project using Live Server or open `index.html` in your browser.

## 🔑 API Configuration

This project uses the OpenWeatherMap API.

Steps:

1. Create an account
2. Generate an API key
3. Replace it in `app.js`
